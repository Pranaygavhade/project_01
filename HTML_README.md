# 🎬 Pranay Gavhade Portfolio - HTML Version

## 🚀 Quick Start (No Installation Required!)

### Option 1: Open Directly
1. Navigate to the `portfolio` folder
2. Double-click `index.html`
3. Your portfolio opens in your default browser! 🎉

### Option 2: Use Live Server (Recommended)
If you have VS Code:
1. Install "Live Server" extension
2. Right-click `index.html`
3. Select "Open with Live Server"
4. Enjoy hot-reload while editing!

## 📁 Files Included

```
portfolio/
├── index.html       ← Main HTML file (OPEN THIS!)

```

## 🎨 Customization Guide

### 1. Update Your Name & Info

**File**: `index.html`

Find and replace:
- Line 52: `<h1 class="hero-name fade-in">Pranay Gavhade</h1>`
- Line 54: `<p>CSE (AI) Student | DSA Learner</p>`
- Line 56: Your tagline

### 2. Update Contact Links

**File**: `index.html`

Find the contact section (around line 250):
```html
<a href="mailto:YOUR_EMAIL@example.com" class="social-card reveal">
<a href="https://github.com/YOUR_USERNAME" target="_blank" class="social-card reveal">
<a href="https://linkedin.com/in/YOUR_USERNAME" target="_blank" class="social-card reveal">
```

### 3. Update Projects

**File**: `index.html`

Find the projects section (around line 180) and update:
- Project titles
- Descriptions
- Tech stack badges
- Add real links to buttons

### 4. Update Skills

**File**: `index.html`

Find the skills section (around line 120) and adjust:
- Skill names
- Percentage values in `style="width: XX%"`

### 5. Change Colors

**File**: `styles.css`

Find these color variables and change them:
```css
#a855f7  ← Purple
#fb923c  ← Orange
#fbbf24  ← Gold
#ec4899  ← Pink
#0d0d0d  ← Dark background
#1a1a1a  ← Dark gray
```

## 🚀 Deployment

### GitHub Pages
1. Create a GitHub repository
2. Upload `index.html`, `styles.css`, `script.js`
3. Go to Settings → Pages
4. Select main branch
5. Your site is live!

### Netlify
1. Go to [netlify.com](https://netlify.com)
2. Drag and drop the portfolio folder
3. Done!

### Any Web Host
Just upload the three files to your hosting:
- index.html

## 🐛 Troubleshooting

### Styles not loading?
- Make sure `styles.css` is in the same folder as `index.html`
- Check the file name is exactly `styles.css`

### JavaScript not working?
- Make sure `script.js` is in the same folder
- Check browser console (F12) for errors

### Custom cursor not showing?
- Custom cursor only works on desktop
- It's hidden on mobile devices

### Animations not smooth?
- Try a different browser
- Check if hardware acceleration is enabled
